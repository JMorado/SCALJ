# SCALeJ

Lennard-Jones parameters fitting via condensed-phase volume scaling.

## Installation

Clone the repository:

```bash
git clone https://github.com/JMorado/SCALeJ.git
cd SCALeJ
```

Install SCALeJ using pixi:

```bash
pixi shell
```

## Usage

Currently, SCALeJ is a work in progress and only the API is provided. A CLI and Snakemake pipeline will be provided in the future.
